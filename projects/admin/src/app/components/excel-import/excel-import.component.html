Import Store Orders Upload and process Excel files with store order data

Please upload an Excel file (.xlsx or .xls) containing the store orders data. The system will parse through the entire file and extract the following information:

  • Routes are identified by "Driver Run" or "Driver Run:" in column A with the route name in column B
  • For each route, store names are found in column A
  • Total quantity per store is found in column D
  • Only stores with a quantity greater than 0 will be included
  • Each route ends when the next "Driver Run" is found or at the end of the file
  • Crates needed is automatically calculated (1 crate = 25 items)
